Announcement: Rent function temporarily closed

Dear players:

Since CryptoGladiator starts public beta, we have found a bug that clone CD would be extended indefinitely when renting gladiator, which has affected the normal experience of the game. We apologize for the following decision: We will temporarily close rent function and start repair work.

Players who have already encountered this bug should please don't be worry and wait patiently for this repair work to be completed. In the next version, we will complete this repair, and your gladiator will return to normal.

In order to ensure the normal repair work,please let your gladiator off the renting shelf so as to avoid unpredictable losses.

We are very sorry for the inconvenience, thank you for your support and understanding!

Official team
